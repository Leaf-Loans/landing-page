Accessibility

WCAG 2.1 AA is the floor. AAA for hero + primary CTAs.

Focus rings

Every interactive element gets border-color: var(--ll-accent-500); box-shadow: 0 0 0 3px var(--ll-accent-100); — no outline: none without replacement.

Touch targets

Minimum 44 × 44 px on mobile. Padding the tap-area is acceptable even if the visual glyph is smaller.

Reduced motion

All reveal animations and transitions honor prefers-reduced-motion: reduce automatically via the global rule in tokens.css.

Semantic HTML

<nav>, <section>, <button>, one <h1> per page. Never use <div> for interactive triggers.
